=========

A small library that adds commas to numbers

## Installation

  `npm install @khalid0807/number-formatter`

## Usage

    var numFormatter = require('@khalid0807/number-formatter');

    var formattedNum = numFormatter(35666);
  
  
  Output should be `35,666`


## Tests
